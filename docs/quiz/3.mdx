## Q3. 다음 중 웹 접근성의 원칙과 설명이 잘못 짝지어진것을 고르세요.

import { ObjectiveQuiz } from "../_util/ObjectiveQuiz.mdx";

  <Correct>
    <p>
      3. 이해 가능 원칙은 모든 사용자가 능력에 관계없이 웹 콘텐츠의 정보뿐 아니라 원하는
      정보를 찾기 위한 탐색 방법 및 유저 인터페이스나 기능을 사용하는 방법을 이해할 수 있어야
      함을 의미합니다.
    </p>
  </Correct>
  <p>
    접근성에는 Perceivable(인식 가능), Understandable(이해 가능), Operable(조작 가능),
    Robust(견고성) 4 가지의 상위 수준 원칙이 있고 이를 줄여서 P.O.U.R 원칙이라고 부릅니다.
  </p>
  <p>
    이 원칙은 접근성에서 다루어야 하는 문제들을 기능적으로 설명하고 이 원칙에 따라 접근성
    지침 표준이 마련되어 있습니다.
  </p>
