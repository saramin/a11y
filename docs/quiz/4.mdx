## Q4. 다음 웹 접근성 표준 지침에 대한 설명 중 잘못된 것을 모두 고르세요.

  <Correct>
    <p>
      3. 최소 기준은 문자 그대로 "최소한의 기준"이기 때문에, 최소 기준을 만족하지 못하면
      장애를 가진 대부분의 사용자가 사용하기 어려운 상태를 유지하게 됩니다. <br />
      예를 들어, 4.5:1 이상의 명도대비 요구사항이 있는 경우 4:1의 명도대비 제공은
      해당 요구사항을 가진 사용자의 문제를 전혀 해결할 수 없게 됩니다.
    </p>
    <p>
      5. 위에 언급된 것과 마찬가지로 최소한의 기준이기 때문에, 사용이 불가능한 수준을 겨우
      넘어선 상태가 되어 여전히 사용이 어려운 사용자 그룹이 남아있게 될 수 있습니다. <br />
      사용자에게 좋은 경험을 제공하기 위해서는 최소 기준을 상회하는 수준의 대응이 필요합니다.
    </p>
  </Correct>
  <p>
    웹 접근성 표준 지침은 장애인이 비장애인과 동등하게 웹 콘텐츠에 접근할 수 있도록 웹
    콘텐츠를 제작하는 방법에 관하여 기술하고 있습니다.
  </p>
  <p>
    월드 와이드 웹 컨소시움(W3C)에서 국제 표준인
    WCAG(Web Content Accessibility Guidelines)를 제정하며, 이 지침을 기반으로
    한국형 웹 콘텐츠 접근성 지침이 국가 표준으로 제정되어 있습니다.
  </p>

    가이드 라인을 수립하였습니다.
